# ✈️ Flight Search App

A modern flight search application built with **Next.js** and powered by the **Amadeus API** to deliver real-time flight data.  
Try the live demo here: [https://flight-search-gold.vercel.app/](https://flight-search-gold.vercel.app/)

## 🚀 Features

- 🔍 **Search Flights**: Find flights between any two destinations with flexible date options.  
- 📅 **Date Picker**: Intuitive calendar UI for selecting departure and return dates.  
- 👥 **Passenger Selector**: Easily adjust the number of adults, children, and infants.  
- 🌍 **Destination Autocomplete**: Smart airport/city suggestions as you type.  
- 📊 **Responsive UI**: Optimized for desktop and mobile devices.  
- ⚡ **Fast & Reliable**: Built with Next.js for server-side rendering and smooth performance.  

## 🛠️ Tech Stack

- **Frontend Framework**: [Next.js](https://nextjs.org/)  
- **UI Components**: React, TailwindCSS (or your chosen styling library)  
- **Forms**: React Hook Form for robust form handling  
- **API Integration**: [Amadeus API](https://developers.amadeus.com/) for flight search and availability  
- **Deployment**: [Vercel](https://vercel.com/)
